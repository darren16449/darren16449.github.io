FinSynGen — LLM-Based Synthetic Financial Time-Series Generator

Python OpenAI Time Series Statistical Validation

Problem: Real market data can be insufficient for stress testing and simulation of rare or extreme scenarios.

Approach: Generate synthetic log-return series using LLM prompting, then automatically validate realism with statistical checks — rejecting and regenerating weak samples.

Prompt Engineering → LLM Generation → KS / Distribution Tests → Auto-Regeneration → Exportable Dataset

Impact: Enables risk simulation and scenario testing when historical datasets are limited or incomplete — no manual data collection required.

Demonstrates: Statistical evaluation (DS), pipeline automation (DE), LLM workflow design (ML).

Marketing Content Assistant — LLM + RAG Application

OpenAI RAG ChromaDB Streamlit

Problem: Marketing content needs speed, consistency, and brand control — but generic LLM outputs drift and hallucinate without grounding.

Approach: Retrieval-Augmented Generation grounds every output in relevant context, with prompt templates enforcing tone and format controls for multilingual generation.

User Input → Embedding Search (ChromaDB) → Context Injection → GPT Generation → Structured Output

Impact: Produces grounded, reusable marketing copy with controllable tone, format, and keyword guidance — reducing content drift significantly.

Demonstrates: RAG implementation (ML), vector search systems (DE), structured outputs (Analytics).
